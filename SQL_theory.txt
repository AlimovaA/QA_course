SQL (Structured Query Language) - Язык структурированных запросов 

База данных - это совокупность данных, которые хранятся по определенным правилам и используются для удовлетворения информационных потребностей пользователей.
Реляционная база данных - это БД, где вся информация содержится в виде таблиц.
Таблица - двумерная структура данных, состоящая и строк-row и столбцов-column.

DML (Data Manipulation Language) - эти команды работают с данными в таблицах (внутренностью таблиц).
Команды: 
SELECT - делает выборку из таблиц, т.е. находит нам то, что нам необходимо.
INSERT - вставлять. С помощью данной команды мы будем вставлять строки в наши таблицы.
UPDATE - с помощью данной команды мы сможем изменять данные в таблицах.
DELETE - удалять. С помощью данной команды мы сможем удалять строки в наших таблицах.
MERGE - объединение. Cпособен объеденить функциональномть INSERT, UPDATE и DELETE.

DDL (Data Definition Language) - напрямую с данными не работают, команды работают с таблицей в целом или со структурой таблицы. 
Команды:
CREATE - команда может создавать таблицы.
ALTER - с помощью этой команды мы можем изменять структуру таблицы, например добавлять новые столбцы.
DROP - удаляет объекты.
RENAME - данная команда позволяет поменять название столбца.
TRUNCATE - удаляет все данные из таблицы, т.е. сама таблица остается, но все, что было в ней - удаляется.

TCL (Transaction Control Language) - контроль транзакций. Команды: 
COMMIT - подтвердить
ROLLBACK - откатить
SAVEPOINT - точка сохранения

DCL (Data Control Language) 
Команды:
GRANT - отдавать
REVOKE - отобрать
С помощью этих команд дают или отбирают права на использование, просмотра таблиц.
